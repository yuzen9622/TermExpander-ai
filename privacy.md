TermExpander AI 隱私權政策 (Privacy Policy)
最近更新日期：2025 年 12 月 21 日

本隱私權政策旨在說明 TermExpander AI（以下簡稱「本擴充功能」）如何收集、使用及處理您的資料。我們非常重視您的隱私，致力於保護您的個人資訊。

1. 我們處理的資料
   本擴充功能僅處理實現其核心功能所需的最小量資料：

API 金鑰 (API Key)： 您所提供的 Google Gemini API 金鑰會透過 chrome.storage.local 安全地儲存在您的本地瀏覽器中。我們不會將您的 API 金鑰上傳到我們的伺服器或任何第三方服務。

選取文字 (User Selected Text)： 當您在網頁上選取專有名詞時，本擴充功能會擷取該段文字並傳送至 Google Gemini API 以獲取完整的學術名稱。

2. 資料的使用方式
   功能實現： 您選取的文字僅用於向 AI 發出請求，以識別並補全學術術語。

本地儲存： 所有的設定（如 API 金鑰、模型選擇）皆儲存於您的裝置本地。

3. 資料的披露與第三方傳輸
   外部 API 請求： 為了提供 AI 功能，您的選取文字會透過加密的 HTTPS 連線傳送至 [Google Gemini API](https://www.google.com/search?q=generativelanguage.googleapis.com)。該資料的處理受 Google 隱私權政策 之約束。

不轉售資料： 我們絕不會向任何第三方出售、轉讓或使用您的資料進行廣告投放。

4. 安全性
   我們使用現代加密技術 (HTTPS) 傳輸所有傳出資料。由於所有敏感資料（如 API Key）皆儲存在您的本地設備中，您可以隨時透過清除瀏覽器資料或解除安裝擴充功能來刪除這些資訊。
